slidesConfigProvider
service in module sdco-slides

Description

Provider of slidesConfig This service can be used to configure some slides behaviors for a new application. The applyConf method has to be called during the configuration phase.

Methods

  • applyConf()

    Do all needed configuration steps such as routing...

  • setTemplatesRootPath(templatesRootPath)

    Update the config object by adding the templatesRootPath property for the root path for your templates

    Parameters
    ParamTypeDetails
    templatesRootPathString

    the root path for your templates

    (default: 'views/partials/')

Example

angular.module('slidesSample', ['sdco-slides'])
//Config routes
.config(['slidesConfigProvider',
  function(slidesConfigProvider){
    slidesConfigProvider.applyConf();
}])
//Init view classes
.run(['slidesConfig',  function(slidesConfig){
  slidesConfig.init();
}]);
