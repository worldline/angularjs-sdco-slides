  • Sometimes, we don't want the compilation to be done each time a key is pressed, use the compile on demand for that. The value compile-on-demand is a parameter of the directive, and can be modified dynamically by the user in the UI.
  • In some contexts, the "hot compilation" doesn't work fine (angular directives for example). In such a case, the user can use the jsFiddle option which will open the editor content in jsFiddle. Use the js-fiddle parameter to display it or not by default, it can still be modified dynamically by the user
