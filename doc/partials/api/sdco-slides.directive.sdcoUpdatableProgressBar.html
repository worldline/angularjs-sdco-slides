sdcoUpdatableProgressBar
directive in module sdco-slides

Description

Used internally only. Add a progress bar describing the advancement of slides. It is the possible to click on the bar to go to a specific slide.

Usage

as element:
<sdco-updatable-progress-bar
       the-array="{Array}"
       current-index="{int}"
       progress-bar-display="{String}">
</sdco-updatable-progress-bar>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
theArrayArray

the array containing slides data. Should be replaced by an integer later.

currentIndexint

the binded index

progressBarDisplayString

'all' if you want to display the slides numbers directly in the bar or 'global' if you want a global pager to be added

(default: global)
