sdcoSlidesGoTo
directive in module sdco-slides

Description

Listen for click on the element it is used on, and go to the slides targeted by 'dest'

Usage

as attribute
<ANY sdco-slides-go-to
     dest="{Integer}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
destInteger

the index of the slide to go to
