slidesConfig
service in module sdco-slides

Description

This service owns some configuration attributes. It's init function has to be called at start.

Methods

  • getConfig()

    Get the config object

    Returns
    Object

    the config object

    Example

    angular.module('slidesSample', ['sdco-slides'])
//Config routes
.config(['slidesConfigProvider',
  function(slidesConfigProvider){
    slidesConfigProvider.applyConf();
}])
//Init view classes
.run(['slidesConfig',  function(slidesConfig){
  slidesConfig.init();
}]);
